# 2026-07-17

## PieTypeMenu 背景模糊改为全屏遮罩
- 原模糊层是以入口为中心的 760px 圆（`backdrop-filter` + 径向 `mask` 渐隐），只模糊扇形附近一小片，圆外（含扇形最外层背景）完全未模糊，用户反馈“最外层背景还是没有模糊的”。
- 改为 `.pie-scrim` 全屏固定遮罩（`position:fixed; inset:0` + `backdrop-filter: blur(7px)`），覆盖整个视口，入口附近叠一层卡片色薄纱（`radial-gradient at 90% 96%`）聚焦扇形；去掉圆形 `mask` 硬边界，最外层背景也被模糊。菜单项 `z-index:2` 仍在其上层保持清晰。
- 文件：`frontend/src/components/PieTypeMenu.vue`（源码改动，dev server 5173 HMR 已生效；未提交，待用户“提交/发版”指令，与上次扇形模糊未提交改动累积）。

## 修正：扇形(圆形)模糊 + 动态覆盖最外圈
- 用户澄清：要的是**扇形区域内**的圆形模糊（首版方向对），不要全屏；上一版问题只是模糊圆太小、mask 在 274px 提前渐隐，导致第三圈(≈304px)落在有效模糊区外没被覆盖。
- 改回圆形模糊并让直径动态：新增 `scrimRadius = max(各扇形项到入口距离) + 70`、`scrimSize = scrimRadius*2`，经 inline style 绑定到 `.pie-scrim` 的 width/height；mask 渐隐推到圆边缘(`#000 74%, transparent 100%`)，使模糊有效区覆盖到最外一圈(含第三圈及以上)；薄纱 `background` 保留聚焦。菜单项 z-index:2 仍清晰。
- 文件：`frontend/src/components/PieTypeMenu.vue`（仍是未提交源码改动，待用户“提交/发版”指令）。
