# 2026-07-20 工作日志

## 首页睡眠记录排序改为按醒来时间（用户需求）
- 需求：首页记录列表，睡眠类型按「醒来时间」排序；没有醒来时间（进行中）按当前时间排序（相当于始终最新）。
- 实现（`frontend/src/views/Home.vue` 的 `buildTimeline` 睡眠分支）：睡眠 item 的 `sortKey` 由 `s.occurred_at`（入睡时间）改为 `s.sleep_end ? s.sleep_end : tsToIso(Date.now())`。整条 timeline 在 `items.sort((a,b)=>b.sortKey.localeCompare(a.sortKey))`（约 683 行）统一以 `sortKey` 降序排列（最新在前），故睡眠按醒来时间定位其在列表中的顺序；进行中睡眠 sortKey=当前时间，恒置顶。`anchorTs`/`gapStartTs`（「距现在 / 距上次」文案计算）不受影响，仅排序键变化；`sleep_end` 与 `occurred_at` 同为 `tsToIso` 格式，字符串比较一致。后端 list 返回顺序由前端重排覆盖，无需改后端。
- 验证：`pnpm build` 通过，重启后端后首页加载新 JS（index-CU4aREKa.js）。代码改动（Home.vue）未提交，待用户指令；记忆文件按约定主动提交。
